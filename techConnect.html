Tech Connect

Full Stack: Angular, NestJS, NodeJS, MySql, Prisma

This web project is a modern, full-stack application developed using Angular for the front end, NestJS for the backend, and Prisma as the database ORM, all running on Node. The primary functionalities of the application include user registration, login. A todo list, user profiles, and a posts feature are currently in development. The project aims to provide a comprehensive user experience, incorporating essential features like user management, task tracking, and social interaction through posts. The technology stack ensures a scalable and maintainable codebase, making it well-suited for future expansions and improvements.
