COVID-19 Global Church Hackathon

React JS

In response to the Coronavirus outbreak, Faith Tech put on a Hackathon challenge. With 10 challenges to choose from and one week until completion, our team of 12 developers created a website that would serve our neighbors with their basic needs. A user creates a profile and can request food and household items they are lacking. The local Church would then step in to provide for the needs of the community. There is also a way to give in order to help supply the needs of others as well. The front end was built with REACT JS using react-bootstrap. The website includes login, password hashing, JWT, user and admin profile views. The back end was built with EXPRESS JS so that users can post their needs to a database. All available items are stored in the database as well as the user and admin profile information.
